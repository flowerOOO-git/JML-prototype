# 开发环境页面复刻项目

## 项目说明

本项目基于你提供的分享包素材，生成了一个可直接运行的前端页面壳，用于复刻展示 `今麦郎BPM平台-开发环境_files` 对应页面。

- 入口：`开发环境页面复刻-分享包/index.html`
- **左侧导航**：首页与工作台各自保留完整左侧导航，切换页面时统一为工作台导航样式（不隐藏首页侧栏）
- 首页：`../今麦郎BPM平台-开发环境.html`
- 工作台：`../今麦郎BPM平台-工作台.html`（左侧菜单点击「工作台」切换）
- 通用申请审批：`../通用申请审批页面-分享包/index.html`（点击流程单号 `TYSX-202605170001` 在新标签页打开）
- 静态资源：`../今麦郎BPM平台-开发环境_files/`、`../今麦郎BPM平台-工作台_files/`

## 本地运行

请在工作区根目录启动静态服务（不要直接双击 HTML）：

```bash
python3 -m http.server 8080
```

然后访问：

[http://localhost:8080/开发环境页面复刻-分享包/index.html](http://localhost:8080/%E5%BC%80%E5%8F%91%E7%8E%AF%E5%A2%83%E9%A1%B5%E9%9D%A2%E5%A4%8D%E5%88%BB-%E5%88%86%E4%BA%AB%E5%8C%85/index.html)
